# Scientific Writing with LaTeX — Practice Repository

This repository contains my hands‑on practice materials from the course **“[Scientific Writing with LaTeX](https://faradars.org/courses/mvrtex9201-academic-writing-latex)”** published on [Faradars.org](https://faradars.org/) and taught by **Dr. Kalami Heris**, whose courses are widely appreciated for their clarity, structure, and high teaching quality.

While following the course, I implemented every concept directly in LaTeX and organized each topic into its own directory. Each folder includes the corresponding source code, small experiments, and examples related to that specific concept.

The result is a structured collection of practical LaTeX snippets that can serve both as a learning record and a quick reference for common techniques used in scientific writing.

### The goal of this repository is to serve as:

- #### A personal reference for LaTeX concepts
- #### A structured set of examples for anyone learning the LaTeX
- #### A clean archive of practical implementations from basic to more advanced topics

## 🌱 Personal Note

I completed this course during a period of significant instability, when daily life in Iran was under war pressure and access to the global internet was fully blocked. For long stretches, only domestic websites were reachable, and many essential online resources were unavailable. These limitations created an environment that was both technically challenging and mentally exhausting.

At that time, I intentionally looked for a learning path that did not rely on stable connectivity. Since my LaTeX setup was already installed locally, it became the perfect choice—something I could study fully offline, allowing me to keep learning even while access to the outside world was very very extremely limited.

Continuing this course became more than just a technical exercise; it was a way to maintain momentum, preserve a sense of normal life, and stay away from overwhelming news cycles and the stressful atmosphere created by ongoing conditions and restrictions. Despite the circumstances, the course itself was excellent, and Dr. Kalami-Heris’s thoughtful and structured teaching made the learning process highly enjoyable.

This was one of four courses I completed during several months of internet blackout. Even as I write this note, only a small portion of international websites are accessible, and according to NetBlocks statistics, Iran has been experiencing 65'th days of internet blackout. Access to GitHub is still possible only through the limited pathways that remain open by this dictator regime.

Actually, I started learning to survive and be well.
